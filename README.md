# גן הגילוי 🌱

> Built in an afternoon with Claude Code. My daughter is now addicted. Parenting win.

An educational PWA for toddlers (ages 1–3). Every tap is a discovery — animals, sounds, colors, shapes. No ads, no accounts, no dark patterns. Just a child and a screen doing something vaguely useful.

## What it is

- Touch-optimized Hebrew-first interface built for tiny, unpredictable fingers
- PWA — installs on home screen, works offline, feels native
- Designed for ages 1–3 which means: large targets, immediate feedback, zero text dependency
- Built with vanilla JS, zero frameworks, zero npm, zero build step

## Why vanilla JS

Because a 2-year-old does not care about your bundle size philosophy, but she does care if the thing loads in under a second on a mid-range Android.

## Tech

Pure HTML + CSS + JS. One `index.html`. Deploys to Netlify in 30 seconds.

# clone and open index.html
# that is literally it
